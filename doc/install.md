# Installation

## Download demonstrator source
1. Switch to home directory `cd ~`
2. Download demonstrator-gui `git clone --recurse-submodules https://git.informatik.uni-rostock.de/mj244/sindabus-demonstrator.git`

## Install dependencies
1. Install python3-kivy `sudo apt install python3-kivy`
2. Install garden (kivy packet manager) `sudo cp sindabus-demonstrator/tools/garden /usr/local/bin`
